## Original pydub Files

The following files in `pydub_plus/core/` are from the original pydub repository:

- `audio_segment.py` - Core AudioSegment class
- `effects.py` - Audio effects
- `exceptions.py` - Exception classes
- `generators.py` - Audio generators
- `logging_utils.py` - Logging utilities
- `playback.py` - Playback functionality
- `pyaudioop.py` - Audio operations
- `scipy_effects.py` - SciPy-based effects
- `silence.py` - Silence detection
- `utils.py` - Utility functions

## Modifications

pydub-plus extends the original pydub with:

- GPU acceleration (CuPy)
- Async processing
- REST API wrapper
- CLI tools
- YouTube/TikTok workflows
- Batch processing

All original pydub functionality remains unchanged and fully compatible.
